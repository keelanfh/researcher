Past projects

The Electionary

I analysed the use of language in American presidential election campaigns from 1960 to the present day, with a group of other students on my Quantitative Methods course at UCL. I used Python and scrapy to collect and process the data from transcripts of presidential election debates, as well as using the Twitter API to collect data from the 2016 candidates' Twitter activity. We then analysed this data using sentiment analysis techniques. Finally, we produced visualisations (mainly using matplotlib), including a visualisation of geospatial data, and produced a website. The code used for the project is available here.

Walking to university

I examined my behaviour walking to and from UCL to see which parts of my journey were faster and slower. I collected location and time data from my phone using Moves. I analysed this using Python, and visualised the results using Mapbox. Here's the report I produced for this project. I plan to upload the code to GitHub soon, but I haven't got round to it yet.
